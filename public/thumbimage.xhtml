thumbimage

  • The Type, Width, Height, and Quality can be specified in the URL parameters. (e.g. /thumbimage/sample.jpg?type=webp;w=320;quality=30)
    Image Type URL parameters
    type (Required) w h quality
    AVIF avif 1-9999 1-9999 1-100 (default: 80)
    WebP webp
    JPEG jpeg
    PNG png ✘ (ignore)
  • If the parameter w or h is larger than the size of the original image, thumbnails will be generated at the size of the original image.
    Original image size URL parameters Generated image size
    1920×1080 w=9999;h=9999 1920×1080
    w=9999;h=540 960×540
    w=360;h=540 480×270
    w=9999 1920×1080
    h=540 960×540
    (not specified) 1920×1080
  • Check the Sec-Fetch-Dest header to prevent generation when URLs are typed directly or embedded from other sites without permission.
    • For browsers that do not support the Sec-Fetch-Dest header, substitute a check in the referer header.
  • If a thumbnail image has already been generated (in the case of the second or later requests), the generated image will be displayed instead of processing the generation.
  • If the original image has been updated (the file's modified date is newer than the thumbnail image), the generation process will be repeated.

Optimal image type selection by using the picture element

<picture>
	<source type="image/avif" srcset="/thumbimage/sample.jpg?type=avif;w=360;quality=30" />
	<source type="image/webp" srcset="/thumbimage/sample.jpg?type=webp;w=360;quality=30" />
	<img src="/thumbimage/sample.jpg?type=jpeg;w=360;quality=30" alt="Sample Image" />
</picture>

Optimal image type selection by using the accept request header

<img src="/thumbimage/sample.jpg?type[]=avif;type[]=webp;type[]=jpeg;w=360;quality=30" alt="Sample Image" />
