thumbimage

Automatically generate thumbnail images.

Sample

  • The Type (JPEG, WebP, AVIF), Width, MaxHeight, and Quality can be specified in the URL parameters.
  • Check the Sec-Fetch-Dest header to prevent generation when URLs are typed directly or embedded from other sites without permission.
    • For browsers that do not support the Sec-Fetch-Dest header, substitute a check in the referer header.
  • If a thumbnail image has already been generated (in the case of the second or later requests), the generated image will be displayed instead of processing the generation.
  • If the original image has been updated (the file's modified date is newer than the thumbnail image), the generation process will be repeated.
